# 🎉 欢迎来到我的博客

这是你的第一篇博客文章！

## 💡 这个文件是什么？

这是一篇示例文章，放在 `source/_posts/` 目录中。你可以：

- ✏️ 编辑这个文件的内容
- 🗑️ 删除它
- 📄 创建你自己的文章

## 📝 如何写文章？

### 快速开始

```bash
# 方式一：使用命令创建（推荐）
hexo new "我的第一篇文章"

# 方式二：手动创建
# 创建文件 source/_posts/my-post.md
```

### 文章格式

每篇文章都需要包含前置元数据（YAML Front Matter）：

```markdown
---
title: 文章标题
date: 2026-04-09
updated: 2026-04-09
tags:
  - 标签1
  - 标签2
categories:
  - 分类
description: 简短描述（显示在列表中）
---

# 文章内容从这里开始

正文内容...
```

## 🚀 本地预览

```bash
# 启动本地服务器
npm run server

# 在浏览器中打开
# http://localhost:4000
```

## 📚 主要特性

- ✅ **快速构建** - 纯静态生成，加载极快
- ✅ **Markdown 支持** - 所有标准 Markdown 语法
- ✅ **分类和标签** - 方便内容组织
- ✅ **SEO 友好** - 自动生成 sitemap 和 feed
- ✅ **主题丰富** - 可选择数百个主题

## 🔧 常用命令

```bash
# 启动开发服务器
npm run server

# 生成静态文件
npm run build

# 清除缓存
npm run clean

# 新建文章
hexo new "文章标题"

# 删除这篇文章
rm source/_posts/welcome-to-my-blog.md
```
